# Реструктуризация кабинета фермера

## Что изменилось

### Вкладка "Моё хозяйство" (diagnosis)
**Было:**
- Местоположение (Регион)
- Ваши владения (Активы)

**Стало:**
- Местоположение (Регион)
- Ваши владения (Активы)
- **+ Что в хозяйстве** (земля в собственности/аренде)
- **+ Сотрудники** (постоянные/сезонные)
- **+ Животные** (КРС, МРС, Птица, Пчеловодство)
- **+ Что высаживаете** (культуры с урожайностью)
- **+ Мой гараж** (техника)

### Вкладка "Профиль" (profile)
**Содержит только личные данные:**
- Название фермы
- Имя
- Фамилия
- Телефон
- Email

### Удалена вкладка "Мой гараж"
- Функционал перенесён в блок "Мой гараж" во вкладке "Моё хозяйство"
- Старые компоненты GarageItemForm, GarageList, TechScoreCard, LeaderboardCard больше не используются на главной странице
- Функции calculateTechScore, handleAddGarageItem, handleRemoveGarageItem, handleSaveGarage удалены

## Структура компонентов

### ExtendedProfile
Декомпозирован на модульные компоненты:
- `profile/types.ts` - общие типы и константы
- `profile/LandSection.tsx` - земля
- `profile/EmployeesSection.tsx` - сотрудники
- `profile/AnimalsSection.tsx` - животные
- `profile/CropsSection.tsx` - культуры
- `profile/EquipmentSection.tsx` - техника

### ProfileEditor
Остался без изменений, содержит только форму для личных данных фермера.

## Преимущества
1. **Всё в одном месте** - вся информация о хозяйстве в одной вкладке
2. **Логическая структура** - "Моё хозяйство" = всё про хозяйство, "Профиль" = личные данные
3. **Упрощённая навигация** - 5 вкладок вместо 6
4. **Модульность** - ExtendedProfile легко поддерживать благодаря декомпозиции

## API интеграция
Все блоки используют единый endpoint:
- URL: `https://functions.poehali.dev/8d27026d-2391-42d4-ac54-57f1cbe0c8d4`
- Методы: GET (загрузка), POST (добавление), DELETE (удаление)
- Actions: save_land, save_employees, add_animal, add_crop, add_equipment

## Система рейтинга
Автоматически пересчитывается при любом изменении данных через ExtendedProfile:
- После сохранения земли
- После добавления животных
- После добавления культур
- После добавления техники
